<br/>

### 1. Install [ESLint][eslint] if you haven't already

<br/>

npm:

```sh
npm i eslint@8.22.0 --save-dev
```

yarn:

```sh
yarn add eslint@8.22.0 -D
```

<br/>

### 2. Install package

<br/>

npm:

```sh
npm -i --save-dev @garvae/eslint-config-garvae
```

yarn:

```sh
yarn add -D @garvae/eslint-config-garvae
```

<br/>

### 3. Add the config to the ESLint config

<br/>

```json
{
  "plugins": ["@garvae/eslint-config"]
}
```
